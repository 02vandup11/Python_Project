# Currency Converter (Python)

This project is a beginner-friendly **Currency Converter** built using Python.  
It supports converting between multiple currencies using **fixed exchange rates** stored in a dictionary.  
The project is created inside a Jupyter Notebook (.ipynb), making it easy for beginners to learn and run.

---

## Features

- Convert between INR, USD, and EUR  
- Simple and clean Python code  
- Beginner-friendly logic  
- Input validation  
- Works inside Jupyter Notebook  

---

## Technologies Used

- Python  
- Jupyter Notebook  

---

## How to Run the Notebook

### Run Locally
1. Install Jupyter Notebook:
   ```bash
   pip install notebook
   ```
2. Open Jupyter:
   ```bash
   jupyter notebook
   ```
3. Open the notebook file:
   ```
   currency_converter.ipynb
   ```

### Run on Google Colab
- Upload your `.ipynb` file to Google Colab  
- Run all cells directly in the browser
