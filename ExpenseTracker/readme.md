# Expense Tracker (Python - Jupyter Notebook)

This project is a simple and beginner-friendly **Expense Tracker** built using Python.  
It allows users to add expenses, store them in a CSV file, and view all saved expenses.  
The project is implemented inside a Jupyter Notebook (.ipynb), making it easy for beginners to understand, run, and modify.

---

## Features

- Add expenses with date, category, amount, and notes  
- Automatically stores data in a CSV file  
- View all expenses in a readable format  
- Beginner-friendly file handling (CSV)  
- Uses functions for clean and modular code  

---

## Technologies Used

- Python  
- Jupyter Notebook  
- CSV file handling  
- datetime module  

---

## How to Run the Notebook

### Run Locally
1. Install Jupyter Notebook:
   ```bash
   pip install notebook
   ```
2. Open Jupyter:
   ```bash
   jupyter notebook
   ```
3. Run the notebook file:
   ```
   expense_tracker.ipynb
   ```

### Run on Google Colab
- Upload your `.ipynb` file to Google Colab  
- Run all cells directly in the browser
